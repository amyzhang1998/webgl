## WebGL 学习笔记

### 基础概念

- **顶点着色器（Vertex Shader）**：处理每个顶点的程序，负责设置 `gl_Position`，决定顶点在裁剪空间中的位置。
- **片断着色器（Fragment Shader）**：处理每个像素片断的程序，负责设置 `gl_FragColor`，决定像素的颜色。
- **缓冲区（Buffer）**：用于向 GPU 传递顶点数据的内存区域。
- **属性变量（Attribute）**：顶点着色器中从缓冲区获取数据的变量，如 `a_position`。
- **精度修饰符**：片断着色器中需要指定浮点数精度，`mediump` 表示中等精度。
